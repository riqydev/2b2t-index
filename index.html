About

This is a player lookup for the minecraft anarchy server 2b2t.

By inputting a minecraft username that has logged into the server at least one time, you can view their stats. Such as when they were last seen, join date, etc.

I made this quickly in order to practice some simple web dev stuff / to derust.

This is only possible thanks to rfresh and his api :]
